🔐 How to get your Token (2 min):

1️⃣ Open GitHub.com → Tap profile picture → Settings

2️⃣ Scroll down → Developer settings → Personal access tokens → Tokens (classic)

3️⃣ Click "Generate new token (classic)"

4️⃣ Give any name, e.g., "My File Manager"

5️⃣ Under Scopes, tick these:
✅ repo (all options)
✅ user (read user info)

6️⃣ Scroll down → Generate token

7️⃣ Copy the token (starts with ghp_...)

8️⃣ Paste it above and tap LOGIN

⚠️ Token is like a password. Keep it secret. You can revoke it anytime.
