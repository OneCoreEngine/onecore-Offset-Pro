How to get your GitHub token:

1. Open GitHub.com and go to Settings.

2. Select Developer settings > Personal access tokens > Tokens (classic).

3. Click Generate new token (classic).

4. Enter a name (e.g., OneCore Manager).

5. Under Scopes, select 'repo' (all) and 'user'.

6. Click Generate token and copy the value (starts with ghp_).

7. Paste the token below and tap Login.

Token is like a password. Keep it secret. You can revoke it anytime.
